# ytags-p-ff
stands for: youtube auto generated subs paraphraser for ffmpeg
### 记得star, 支持下

## 介绍:
这是一个能让你用yt-dlp下载好youtube自动生成的字幕(.srt)后给字幕时间轴重组, 这样可以让你用ffmpeg嵌入硬字幕的时候可以只有一行. 软字幕的话还没测试过(理论上可行).

## 要求
##### python ≥ 3.0
##### ffmpeg 都行 (我用的是8.0.1)

## 使用方式:
无需下载
##### 1.先cd到需要处理的文件夹
##### 2.执行此命令:
    curl -s https://raw.githubusercontent.com/ChenChenTown1/ysiol-ec/refs/heads/main/fix_srt.py | python3 -

##### 3.等待处理完毕, 带有_fixed是处理完的
##### 4.用另一个python脚本进行合成
