# ytags-p-ff
Stands for:
Youtube auto generated subs paraphraser for ffmpeg
### 记得star, 支持下

## 介绍:
这是一个能让你用yt-dlp下载好youtube自动生成的字幕(.srt)后给字幕时间轴重组, 这样可以让你用ffmpeg嵌入硬字幕的时候可以只有一行. 软字幕的话还没测试过.

## 使用方式:
    curl -s https://raw.githubusercontent.com/ChenChenTown1/ysiol-ec/refs/heads/main/fix_srt.py | python3 -
